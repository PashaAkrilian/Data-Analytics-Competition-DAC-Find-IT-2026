# Data Analytics Competition DAC Find IT 2026

An end-to-end **data analytics competition project** for **DAC (Data Analytics Competition) Find IT 2026**, focused on transforming raw datasets into **business insights, strategic recommendations, and data-driven storytelling** through exploratory analysis, visualization, and predictive analytics. The current repository contains a dedicated Jupyter Notebook workflow with **15 commits of active development**, making it a strong portfolio-ready competition project. ([github.com](https://github.com/PashaAkrilian/Data-Analytics-Competition-DAC-Find-IT-2026))

## Project Overview

This repository documents the full analytics workflow for the **Find IT 2026 Data Analytics Competition**.

## Problem Statement

The challenge focuses on analyzing a provided competition dataset to uncover:

* trends
* anomalies
* correlations
* KPI movement
* user/customer behavior
* business opportunities
* risk factors

The expected output typically includes:

* data insights report
* visual storytelling
* strategic recommendations
* optional predictive model support

This makes the project highly suitable for **Business Intelligence and Data Analytics portfolios**.

---

## Methodology

### 1) Data Understanding

* inspect schema and column types
* identify business entities
* understand KPI definitions
* map analytical objectives

### 2) Data Cleaning

* missing value handling
* duplicate checks
* outlier inspection
* datatype correction
* normalization of inconsistent labels

### 3) Exploratory Data Analysis

* descriptive statistics
* distribution analysis
* trend analysis
* segmentation analysis
* cohort or category comparisons
* anomaly spotting

### 4) Visualization & Storytelling

Possible visual outputs:

* KPI trend lines
* category comparisons
* heatmaps
* correlation matrix
* customer segmentation charts
* Pareto charts
* contribution analysis

### 5) Strategic Insights

This is the strongest section for DAC competitions.

Focus on:

* what happened
* why it happened
* what should stakeholders do next
* measurable improvement opportunities
